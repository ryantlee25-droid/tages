# Product Hunt Launch

## Tagline
Persistent codebase memory for AI coding agents

## Description (260 chars)
Tages gives AI tools like Claude Code, Cursor, and Codex persistent memory about your codebase. Conventions, decisions, architecture — remembered across sessions via MCP. Open source, local-first, free tier forever.

## First Comment (Founder Story)
Hi PH! I'm Ryan, and I built Tages because I was tired of re-explaining my codebase to AI every single session.

Every time I started a new Claude Code or Cursor session, the agent would re-discover the same architecture, violate the same naming conventions, and suggest solutions I'd already rejected. The context amnesia was costing me hours per week.

Tages is an MCP server that sits between your codebase and your AI tools. It remembers everything — conventions, past decisions, architecture notes, lessons learned — and injects it into every new session automatically.

It's open source, works with any MCP-compatible tool, and has a free tier that's genuinely useful (10,000 memories, all features including auto-indexing). Pro is $9/mo for teams, unlimited projects, and cloud sync. You can also self-host for free.

I'd love your feedback on what memory types would be most useful for your workflow!
